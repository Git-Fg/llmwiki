# Karpathy Wiki Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a markdown-based personal knowledge base CLI in Rust, with NVIDIA NIM embeddings, agent skill integration, and a Svelte static viewer — replacing the gutted `lucasastorian/llmwiki` Python codebase.

**Architecture:** Single Rust crate `wiki` at the repo root. Markdown wiki files + JSONL embeddings (gitignored, regenerated per device). 5-line SKILL.md stub generated by `build.rs`, full skill content embedded in binary via `include_str!` and served by `wiki skill show`. SvelteKit static viewer reads wiki at build time.

**Tech Stack:** Rust 1.85+, clap, tokio, reqwest, serde_yaml, serde_json, pulldown-cmark, notify, walkdir, anyhow, thiserror, tracing. SvelteKit 2 + adapter-static + marked. NVIDIA NIM API.

**Spec:** `docs/superpowers/specs/2026-06-21-karpathy-wiki-design.md`

```rust
use std::path::{Path, PathBuf};
 crate::error::WikiError;

/// Discover the wiki workspace root by priority order:
/// 1. `--workspace <path>` flag
/// 2. `WIKI_WORKSPACE` env var
/// 3. Walk up from CWD looking for `.wiki/` directory
/// 4. `~/wiki` if it exists
pub fn discover_workspace(
    flag: Option<PathBuf>,
    env: Option<PathBuf>,
    cwd: PathBuf,
) -> Result<PathBuf, WikiError> {
    if let Some(p) = flag {
        return Ok(p.canonicalize().unwrap_or(p));
    }
    if let Some(p) = env {
        return Ok(p.canonicalize().unwrap_or(p));
    }
    if let Some(p) = walk_up_for_dot_wiki(&cwd) {
        return Ok(p);
    }
    if let Some(home) = home_dir() {
        let candidate = home.join("wiki");
        if candidate.join(".wiki").exists() {
            return Ok(candidate.canonicalize().unwrap_or(candidate));
        }
    }
    Err(WikiError::WorkspaceNotFound)
}

fn walk_up_for_dot_wiki(start: &Path) -> Option<PathBuf> {
    let mut current = start.canonicalize().ok()?;
    loop {
        if current.join(".wiki").exists() {
            return Some(current);
        }
        if !current.pop() {
            return None;
        }
    }
}

fn home_dir() -> Option<PathBuf> {
    std::env::var_os("HOME").map(PathBuf::from)
}
```

```rust
pub fn load_config(paths: &[PathBuf]) -> Result<Config, WikiError> {
    let mut merged: Config = Config::default();
    for path in paths {
        if !path.exists() {
            continue;
        }
        let text = std::fs::read_to_string(path)?;
        let partial: Config = serde_yaml::from_str(&text).map_err(|e| WikiError::ConfigInvalid {
            path: path.display().to_string(),
            line: e.location().map(|l| l.line()).unwrap_or(0),
            message: e.to_string(),
        })?;
        merged = merge(merged, partial);
    }

    // Validate that the configured embedding/rerank model is whitelisted
    let whitelisted = [
        "nvidia/nv-embed-v1",
        "nvidia/nv-embedqa-e5-v5",
        "nvidia/nv-embedcode-7b-v1",
        "nvidia/llama-nemotron-embed-1b-v2",
        "nvidia/llama-nemotron-embed-vl-1b-v2",
        "nvidia/llama-nemotron-rerank-1b-v2",
        "nvidia/llama-nemotron-rerank-vl-1b-v2",
        "nvidia/nv-rerankqa-mistral-4b-v3",
    ];
    if !merged.nim.embed_model.is_empty() && !whitelisted.contains(&merged.nim.embed_model.as_str()) {
        return Err(WikiError::ConfigInvalid {
            path: "validation".into(),
            line: 0,
            message: format!("Unsupported embedding model: {}", merged.nim.embed_model),
        });
    }

    Ok(merged)
}

fn merge(mut base: Config, over: Config) -> Config {
    if !over.nim.embed_model.is_empty() { base.nim.embed_model = over.nim.embed_model; }
    if !over.nim.rerank_model.is_empty() { base.nim.rerank_model = over.nim.rerank_model; }
    if over.nim.embed_dim_override.is_some() { base.nim.embed_dim_override = over.nim.embed_dim_override; }
    base.config_version = over.config_version;
    base
}

pub fn resolve_config(workspace: &Path) -> Result<Config, WikiError> {
    let mut paths = vec![home_dir().map(|h| h.join(".config/wiki/config.yaml"))];
    paths.push(Some(workspace.join(".wiki/config.yaml")));
    let paths: Vec<PathBuf> = paths.into_iter().flatten().collect();
    load_config(&paths)
}

fn home_dir() -> Option<PathBuf> { std::env::var_os("HOME").map(PathBuf::from) }
```

```rust
pub fn run(path: PathBuf) -> Result<(), WikiError> {
    let target = if path.exists() && path.is_dir() { path.clone() } else {
        std::fs::create_dir_all(&path).with_context(|| format!("create {}", path.display()))?;

    };

    std::fs::create_dir_all(target.join("raw/articles")).context("create raw/articles/")?;
```

```rust
use serde_yaml::Value;
use crate::error::WikiError;

#[derive(Debug, Clone)]
pub struct ParsedPage {
    pub frontmatter: Value,
    pub body: String,
}

pub fn parse_frontmatter(content: &str) -> Result<ParsedPage, WikiError> {
    let trimmed = content.trim_start_matches('\u{feff}'); // strip BOM
    if !trimmed.starts_with("---") {
        return Ok(ParsedPage { frontmatter: Value::Null, body: content.to_string() });
    }
    // Find closing ---
    let after_first = &trimmed[3..];
    let after_first = after_first.trim_start_matches('\n');
    let end = after_first.find("\n---").ok_or_else(|| WikiError::Other(anyhow::anyhow!("unclosed frontmatter")))?;
    let yaml_text = &after_first[..end];
    let body_start = end + 4; // skip \n---
    let body = after_first[body_start..].trim_start_matches('\n').to_string();

    let frontmatter: Value = if yaml_text.trim().is_empty() {
        Value::Null
    } else {
        serde_yaml::from_str(yaml_text)?
    };
    Ok(ParsedPage { frontmatter, body })
}
```

```rust
use serde::{Deserialize, Serialize};

#[derive(Debug, Clone, Serialize, Deserialize, PartialEq)]
pub struct Chunk {
    pub index: usize,
    pub content: String,
    pub start_char: usize,
    pub token_count: usize,
    pub header_breadcrumb: String,
}

pub fn chunk_text(content: &str, chunk_size: usize, overlap: usize, min_tokens: usize) -> Vec<Chunk> {
    if content.trim().is_empty() {
        return vec![];
    }

    let paragraphs: Vec<&str> = content.split("\n\n").map(|p| p.trim()).filter(|p| !p.is_empty()).collect();
    let mut chunks: Vec<Chunk> = vec![];
    let mut current_blocks: Vec<&str> = vec![];
    let mut current_tokens = 0usize;
    let mut current_start = 0usize;
    let mut char_pos = 0usize;
    let mut header_stack: Vec<(usize, String)> = vec![];

    for para in paragraphs {
        let para_tokens = estimate_tokens(para);
        if para.starts_with("#") {
            let level = para.chars().take_while(|&c| c == '#').count();
            let heading = para.trim_start_matches('#').trim().to_string();
            header_stack.retain(|&(l, _)| l < level);
            header_stack.push((level, heading));
        }

        if current_tokens + para_tokens > chunk_size && !current_blocks.is_empty() {
            let text = current_blocks.join("\n\n");
            if estimate_tokens(&text) >= min_tokens {
                chunks.push(Chunk {
                    index: chunks.len(),
                    content: text,
                    start_char: current_start,
                    token_count: estimate_tokens(&text),
                    header_breadcrumb: header_stack.iter().map(|(_, t)| t.as_str()).collect::<Vec<_>>().join(" > "),
                });
            }
            let (overlap_blocks, overlap_tokens) = get_overlap(&current_blocks, overlap);
            current_blocks = overlap_blocks;
            current_tokens = overlap_tokens;
            current_start = char_pos;
        }

        current_blocks.push(para);
        current_tokens += para_tokens;
        char_pos += para.len() + 2;
    }

    if !current_blocks.is_empty() {
        let text = current_blocks.join("\n\n");
        if estimate_tokens(&text) >= min_tokens {
            chunks.push(Chunk {
                index: chunks.len(),
                content: text,
                start_char: current_start,
                token_count: estimate_tokens(&text),
                header_breadcrumb: header_stack.iter().map(|(_, t)| t.as_str()).collect::<Vec<_>>().join(" > "),
            });
        }
    }

    chunks
}

fn estimate_tokens(text: &str) -> usize {
    std::cmp::max(1, text.len() / 4)
}

fn get_overlap(blocks: &[&str], target_tokens: usize) -> (Vec<&str>, usize) {
    let mut result = vec![];
    let mut tokens = 0;
    for block in blocks.iter().rev() {
        let bt = estimate_tokens(block);
        if tokens + bt > target_tokens { break; }
        result.insert(0, block);
        tokens += bt;
    }
    (result, tokens)
}
```

```rust
use serde::{Deserialize, Serialize};
use std::time::Duration;
use crate::error::WikiError;

#[derive(Debug, Clone)]
pub struct NimClient {
    base_url: String,
    api_key: String,
    max_attempts: u32,
    backoff_ms: u64,
    timeout_secs: u64,
    http: reqwest::Client,
}

#[derive(Serialize)]
struct EmbedRequest<'a> {
    input: Vec<&'a str>,
    model: &'a str,
    input_type: &'a str,
}

#[derive(Deserialize)]
struct EmbedResponse {
    data: Vec<EmbedItem>,
}

#[derive(Deserialize)]
struct EmbedItem {
    embedding: Vec<f32>,
}

impl NimClient {
    pub fn new(base_url: String, api_key: String) -> Self {
        let http = reqwest::Client::builder()
            .timeout(Duration::from_secs(30))
            .build().unwrap();
        NimClient { base_url, api_key, max_attempts: 3, backoff_ms: 500, timeout_secs: 30, http }
    }

    pub fn with_max_attempts(mut self, n: u32) -> Self { self.max_attempts = n; self }
    pub fn with_backoff_ms(mut self, ms: u64) -> Self { self.backoff_ms = ms; self }

    pub async fn embed(&self, texts: &[&str], model: &str, input_type: &str) -> Result<Vec<Vec<f32>>, WikiError> {
        if self.api_key.is_empty() {
            return Err(WikiError::NimApiKeyMissing);
        }
        let url = format!("{}/v1/embeddings", self.base_url);
        let body = EmbedRequest { input: texts.to_vec(), model, input_type };

        let mut attempt = 0;
        loop {
            attempt += 1;
            let resp = self.http.post(&url)
                .bearer_auth(&self.api_key)
                .json(&body)
                .send().await
                .map_err(|e| WikiError::NimUnreachable(e.to_string()))?;

            if resp.status().is_success() {
                let parsed: EmbedResponse = resp.json().await.map_err(WikiError::Other)?;
                return Ok(parsed.data.into_iter().map(|i| i.embedding).collect());
            }
            if resp.status().as_u16() == 401 {
                return Err(WikiError::NimUnreachable("401 Unauthorized — check NVIDIA_NIM_API_KEY".into()));
            }
            if attempt >= self.max_attempts {
                return Err(WikiError::NimUnreachable(format!("status {}", resp.status())));
            }
            tokio::time::sleep(Duration::from_millis(self.backoff_ms * attempt as u64)).await;
        }
    }
}
```

```rust
use serde::{Deserialize, Serialize};
use std::path::Path;
use crate::error::WikiError;

#[derive(Debug, Clone, Serialize, Deserialize, PartialEq)]
pub struct ChunkEmbed {
    pub start: usize,
    pub end: usize,
    pub tokens: usize,
    pub embedding: Vec<f32>,
}

#[derive(Debug, Clone, Serialize, Deserialize, PartialEq)]
pub struct PageEmbedding {
    pub path: String,
    pub sha256: String,
    pub model: String,
    pub dim: usize,
    pub chunked: bool,
    pub chunks: Vec<ChunkEmbed>,
    pub embedded_at: String,
}

#[derive(Debug, Clone, Serialize, Deserialize, Default)]
pub struct EmbeddingsFile {
    pub pages: Vec<PageEmbedding>,
}

impl EmbeddingsFile {
    pub fn read_from(path: &Path) -> Result<Self, WikiError> {
        if !path.exists() {
            return Ok(Self::default());
        }
        let text = std::fs::read_to_string(path)?;
        let mut pages = vec![];
        for line in text.lines() {
            if line.trim().is_empty() { continue; }
            let page: PageEmbedding = serde_json::from_str(line)?;
            pages.push(page);
        }
        Ok(EmbeddingsFile { pages })
    }

    pub fn write_to(&self, path: &Path) -> Result<(), WikiError> {
        if let Some(parent) = path.parent() {
            std::fs::create_dir_all(parent)?;
        }
        let tmp = path.with_extension("jsonl.tmp");
        let mut text = String::new();
        for page in &self.pages {
            text.push_str(&serde_json::to_string(page)?);
            text.push('\n');
        }
        std::fs::write(&tmp, text)?;
        std::fs::rename(&tmp, path)?;
        Ok(())
    }

    pub fn find_page(&self, path: &str, model: &str) -> Option<&PageEmbedding> {
        self.pages.iter().find(|p| p.path == path && p.model == model)
    }

    pub fn remove_page(&mut self, path: &str) {
        self.pages.retain(|p| p.path != path);
    }
}
```

```rust
pub fn cosine_similarity(a: &[f32], b: &[f32]) -> f32 {
    if a.len() != b.len() || a.is_empty() {
        return 0.0;
    }
    let dot: f32 = a.iter().zip(b.iter()).map(|(x, y)| x * y).sum();
    let norm_a: f32 = a.iter().map(|x| x * x).sum::<f32>().sqrt();
    let norm_b: f32 = b.iter().map(|x| x * x).sum::<f32>().sqrt();
    if norm_a == 0.0 || norm_b == 0.0 {
        return 0.0;
    }
    dot / (norm_a * norm_b)
}
```

```rust
use std::path::PathBuf;
use sha2::{Sha256, Digest};
use chrono::Utc;
use crate::core::config::resolve_config;
use crate::core::chunker::chunk_text;
use crate::core::embeddings::{EmbeddingsFile, PageEmbedding, ChunkEmbed};
use crate::core::nim::NimClient;
use crate::core::models_registry::{load_registry, Role};
use crate::core::workspace::discover_workspace;
use crate::error::WikiError;

pub struct EmbedArgs {
    pub workspace: Option<PathBuf>,
    pub model: Option<String>,
    pub dims: Option<usize>,
    pub skip_existing: bool,
    pub batch_size: Option<usize>,
}

pub async fn run(args: EmbedArgs) -> Result<(), WikiError> {
    let ws = discover_workspace(
        args.workspace.clone(),
        std::env::var("WIKI_WORKSPACE").ok().map(PathBuf::from),
        std::env::current_dir()?,
    )?;
    let cfg = resolve_config(&ws)?;
    let model_name = args.model.unwrap_or(cfg.nim.embed_model.clone());
    let model_info = load_registry().into_iter().find(|m| m.name == model_name && m.role == Role::Embed)
        .ok_or_else(|| WikiError::Other(anyhow::anyhow!("unknown embed model: {}", model_name)))?;
    let api_key = std::env::var(&cfg.nim.api_key_env).unwrap_or_default();
    let client = NimClient::new(cfg.nim.base_url.clone(), api_key);

    let wiki_dir = ws.join("wiki");
    let jsonl_path = ws.join("embeddings.jsonl");
    let mut emb_file = EmbeddingsFile::read_from(&jsonl_path)?;

    let mut pages: Vec<PathBuf> = vec![];
    for entry in walkdir::WalkDir::new(&wiki_dir) {
        let e = entry?;
        if e.path().extension().and_then(|s| s.to_str()) == Some("md") {
            pages.push(e.path().to_path_buf());
        }
    }

    let mut updated = 0;
    for page_path in pages {
        let rel = page_path.strip_prefix(&ws).unwrap().to_string_lossy().replace('\\', "/");
        let content = std::fs::read_to_string(&page_path)?;
        let sha = format!("{:x}", Sha256::digest(content.as_bytes()));
        if args.skip_existing {
            if let Some(existing) = emb_file.find_page(&rel, &model_name) {
                if existing.sha256 == sha { continue; }
            }
        }
        let chunks = chunk_text(&content, cfg.wiki.default_chunk_tokens, cfg.wiki.chunk_overlap_tokens, cfg.wiki.min_chunk_tokens);
        if chunks.is_empty() { continue; }

        // Batch embed
        let texts: Vec<&str> = chunks.iter().map(|c| c.content.as_str()).collect();
        let embeddings = client.embed(&texts, &model_name, "passage").await?;

        let chunk_embs: Vec<ChunkEmbed> = chunks.iter().zip(embeddings.iter()).map(|(c, e)| ChunkEmbed {
            start: c.start_char, end: c.start_char + c.content.len(),
            tokens: c.token_count,
            embedding: e.clone(),
        }).collect();

        emb_file.remove_page(&rel);
        emb_file.pages.push(PageEmbedding {
            path: rel.clone(),
            sha256: sha,
            model: model_name.clone(),
            dim: model_info.dim,
            chunked: chunk_embs.len() > 1,
            chunks: chunk_embs,
            embedded_at: Utc::now().to_rfc3339(),
        });
        updated += 1;
    }

    emb_file.write_to(&jsonl_path)?;
    println!("✓ Embedded {} page(s)", updated);
    Ok(())
}
```

```rust
pub struct QueryArgs {

#[derive(Serialize)]
struct ChatRequest<'a> {
    model: &'a str,
    messages: Vec<ChatMessage<'a>>,
    max_tokens: u32,
    temperature: f32,
}

#[derive(Serialize)]
struct ChatMessage<'a> { role: &'a str, content: &'a str }

#[derive(Deserialize)]
struct ChatResponse { choices: Vec<ChatChoice> }
#[derive(Deserialize)]
struct ChatChoice { message: ChatMessageOwned }
#[derive(Deserialize)]
struct ChatMessageOwned { content: String }

pub async fn run(args: QueryArgs) -> Result<(), WikiError> {
    let ws = discover_workspace(
        args.workspace.clone(),
        std::env::var("WIKI_WORKSPACE").ok().map(PathBuf::from),
        std::env::current_dir()?,
    )?;
    let cfg = resolve_config(&ws)?;
    let model = args.model.unwrap_or(cfg.nim.embed_model.clone());
    let llm_model = args.llm_model.unwrap_or_else(|| "meta/llama-3.1-70b-instruct".into());

    let jsonl_path = ws.join("embeddings.jsonl");
    let emb_file = EmbeddingsFile::read_from(&jsonl_path)?;
    if emb_file.pages.is_empty() { return Err(WikiError::NoEmbeddings); }

    let api_key = std::env::var(&cfg.nim.api_key_env).unwrap_or_default();
    let client = NimClient::new(cfg.nim.base_url.clone(), api_key);
    let query_vec = client.embed(&[args.question.as_str()], &model, "query").await?;
    let q = &query_vec[0];

    let mut scored: Vec<(String, f32, String)> = vec![];
    for page in &emb_file.pages {
        if page.model != model { continue; }
        for chunk in &page.chunks {
            let score = cosine_similarity(q, &chunk.embedding);
            if score >= 0.2 {
                scored.push((page.path.clone(), score, chunk.embedding.iter().map(|f| f.to_string()).collect::<Vec<_>>().join(",")));
            }
        }
    }
    scored.sort_by(|a, b| b.1.partial_cmp(&a.1).unwrap_or(std::cmp::Ordering::Equal));
    scored.truncate(args.top_k);

    // Build context from chunks
    let mut context_parts: Vec<String> = vec![];
    let mut citation_paths: Vec<String> = vec![];
    for (i, (path, _score, _)) in scored.iter().enumerate() {
        let content = std::fs::read_to_string(ws.join(path)).unwrap_or_default();
        context_parts.push(format!("[{}] {}:\n{}\n", i+1, path, &content[..content.len().min(500)]));
        citation_paths.push(path.clone());
    }
    let context = context_parts.join("\n");

    let system = "You are answering questions about the user's personal knowledge base. Use ONLY the provided context. Cite with [^N] footnote markers.";
    let user = format!("Context:\n{}\n\nQuestion: {}\n\nAnswer with citations.", context, args.question);

    let chat = ChatRequest {
        model: &llm_model,
        messages: vec![
            ChatMessage { role: "system", content: system },
            ChatMessage { role: "user", content: &user },
        ],
        max_tokens: 1024,
        temperature: 0.2,
    };

    let url = format!("{}/v1/chat/completions", cfg.nim.base_url);
    let resp = client.http()
        .post(&url)
        .bearer_auth(std::env::var(&cfg.nim.api_key_env).unwrap_or_default())
        .json(&chat)
        .send().await.map_err(|e| WikiError::NimUnreachable(e.to_string()))?;
    let parsed: ChatResponse = resp.json().await.map_err(WikiError::Other)?;
    let answer = parsed.choices.first().map(|c| c.message.content.clone()).unwrap_or_default();

    if args.json {
        let out = serde_json::json!({
            "question": args.question,
            "answer": answer,
            "citations": citation_paths,
        });
        println!("{}", serde_json::to_string_pretty(&out).unwrap());
    } else {
        println!("\n{}\n", answer);
        if !args.no_citations && !citation_paths.is_empty() {
            println!("\nSources:");
            for (i, p) in citation_paths.iter().enumerate() {
                println!("  [{}] {}", i+1, p);
            }
        }
    }
    Ok(())
}
```

```rust
use crate::core::markdown::parse_frontmatter;
use super::LintIssue;
use super::tag_tax;

const VALID_TYPES: &[&str] = &["entity", "concept", "comparison", "query", "summary"];

pub fn check_frontmatter(path: &str, content: &str) -> Vec<LintIssue> {
    let mut issues = vec![];

    if !super::validate_filename(path) {
        issues.push(LintIssue {
            severity: "error".into(),
            code: "bad-filename".into(),
            path: path.into(),
            message: "filename must be lowercase-hyphenated, e.g. `my-page.md`".into(),
        });
    }

    let parsed = match parse_frontmatter(content) {
        Ok(p) => p,
        Err(_) => {
            issues.push(LintIssue {
                severity: "error".into(),
                code: "missing-frontmatter".into(),
                path: path.into(),
                message: "no YAML frontmatter found (must start with `---`)".into(),
            });
            return issues;
        }
    };

    let fm = match parsed.frontmatter.as_mapping() {
        Some(m) => m,
        None => {
            issues.push(LintIssue {
                severity: "error".into(),
                code: "missing-frontmatter".into(),
                path: path.into(),
                message: "frontmatter block is empty".into(),
            });
            return issues;
        }
    };

    // Required fields per MyWiki/SCHEMA.md
    for field in &["schema_version", "title", "created", "updated", "type", "tags", "sources"] {
        if !fm.contains_key(*field) {
            issues.push(LintIssue {
                severity: "error".into(),
                code: format!("missing-{}", field),
                path: path.into(),
                message: format!("frontmatter missing required field `{}`", field),
            });
        }
    }

    // Validate type value
    if let Some(type_val) = fm.get("type").and_then(|v| v.as_str()) {
        if !VALID_TYPES.contains(&type_val) {
            issues.push(LintIssue {
                severity: "error".into(),
                code: "invalid-type".into(),
                path: path.into(),
                message: format!("type `{}` must be one of: {}", type_val, VALID_TYPES.join(", ")),
            });
        }
    }

    // Validate tags against taxonomy + duplicate detection
    if let Some(tags) = fm.get("tags").and_then(|v| v.as_sequence()) {
        let tag_strs: Vec<String> = tags.iter()
            .filter_map(|v| v.as_str().map(|s| s.to_string()))
            .collect();

        // Duplicate / case-variant check
        let dup_errors = tag_tax::validate_tag_list(&tag_strs);
        for msg in dup_errors {
            issues.push(LintIssue {
                severity: "error".into(),
                code: "duplicate-tag".into(),
                path: path.into(),
                message: msg,
            });
        }

        // Taxonomy membership check
        for tag in &tag_strs {
            if !tag_tax::valid_tag(tag) {
                issues.push(LintIssue {
                    severity: "error".into(),
                    code: "unknown-tag".into(),
                    path: path.into(),
                    message: format!("tag `{}` not in taxonomy (add to SCHEMA.md first)", tag),
                });
            }
        }
    }

    // sources must be non-empty list
    if let Some(sources) = fm.get("sources") {
        if let Some(seq) = sources.as_sequence() {
            if seq.is_empty() {
                issues.push(LintIssue {
                    severity: "warn".into(),
                    code: "empty-sources".into(),
                    path: path.into(),
                    message: "sources list is empty (every page should reference at least one raw source)".into(),
                });
            }
        }
    }

    issues
}
```

```rust
use crate::core::markdown::extract_wikilinks;
use super::LintIssue;

/// Minimum outbound [[wikilinks]] required per MyWiki convention.
const MIN_OUTBOUND: usize = 2;

/// Pages exceeding this line count are candidates for splitting.
const MAX_LINES: usize = 200;

/// Check wikilink health, orphan status, and page-size for a single page.
///
/// `inbound_count` — how many OTHER pages link to this page (pre-computed by the caller).
/// `line_count` — total line count of the file (caller provides from `lines().count()`).
pub fn check_wikilinks(
    path: &str,
    body: &str,
    all_pages: &[String],
    inbound_count: usize,
    line_count: usize,
) -> Vec<LintIssue> {
    let mut issues = vec![];
    let links = extract_wikilinks(body);

    // Build a set of valid slugs (file_stem, lowercase)
    let page_set: std::collections::HashSet<String> = all_pages.iter()
        .filter_map(|p| std::path::Path::new(p).file_stem().and_then(|s| s.to_str()))
        .map(|s| s.to_lowercase())
        .collect();

    // 1. Dangling links — target doesn't resolve to any wiki page
    for link in &links {
        let target = link.split('/').last().unwrap_or(link).to_lowercase();
        if !page_set.contains(&target) {
            issues.push(LintIssue {
                severity: "error".into(),
                code: "dangling-link".into(),
                path: path.into(),
                message: format!("[[{}]] does not resolve to a wiki page", link),
            });
        }
    }

    // 2. Outbound link count — MyWiki requires ≥ 2
    if links.is_empty() {
        issues.push(LintIssue {
            severity: "error".into(),
            code: "no-outbound-links".into(),
            path: path.into(),
            message: "page has 0 outbound [[wikilinks]] (minimum 2 required)".into(),
        });
    } else if links.len() < MIN_OUTBOUND {
        issues.push(LintIssue {
            severity: "error".into(),
            code: "below-min-outbound".into(),
            path: path.into(),
            message: format!(
                "page has {} outbound [[wikilinks]] (minimum {} required)",
                links.len(),
                MIN_OUTBOUND
            ),
        });
    }

    // 3. Orphan detection — no other page links to this one
    if inbound_count == 0 {
        issues.push(LintIssue {
            severity: "warn".into(),
            code: "orphan-page".into(),
            path: path.into(),
            message: "no inbound [[wikilinks]] from any other page (orphan)".into(),
        });
    }

    // 4. Page size — candidate for splitting when too long
    if line_count > MAX_LINES {
        issues.push(LintIssue {
            severity: "warn".into(),
            code: "page-too-long".into(),
            path: path.into(),
            message: format!(
                "page is {} lines (> {}); consider splitting into sub-topics",
                line_count, MAX_LINES
            ),
        });
    }

    issues
}
```

```rust
use std::collections::{HashMap, HashSet};
use std::path::{Path, PathBuf};
use std::io::BufRead;
use sha2::{Sha256, Digest};
use crate::core::markdown::{parse_frontmatter, extract_wikilinks};
use crate::core::workspace::discover_workspace;
use crate::lint::{frontmatter::check_frontmatter, wikilinks::check_wikilinks, LintIssue};
use crate::error::WikiError;

pub struct LintArgs {
    pub workspace: Option<PathBuf>,
    pub scope: String,     // "wiki" | "raw" | "all"
    pub strict: bool,
    pub fix: bool,
    pub json: bool,
}

pub fn run(args: LintArgs) -> Result<(), WikiError> {
    let ws = discover_workspace(
        args.workspace.clone(),
        std::env::var("WIKI_WORKSPACE").ok().map(PathBuf::from),
        std::env::current_dir()?,
    )?;

    let mut all_issues: Vec<LintIssue> = vec![];

    // ── Wiki page lint ──
    if args.scope == "wiki" || args.scope == "all" {
        let wiki_dir = ws.join("wiki");
        if wiki_dir.exists() {
            let mut all_pages: Vec<String> = vec![];
            for entry in walkdir::WalkDir::new(&wiki_dir) {
                let e = entry?;
                if e.path().extension().and_then(|s| s.to_str()) == Some("md") {
                    let rel = e.path().strip_prefix(&ws).unwrap().to_string_lossy().replace('\\', "/");
                    all_pages.push(rel);
                }
            }

            // Pre-compute inbound link counts (orphan detection)
            let inbound = compute_inbound_links(&ws, &all_pages);

            // Read index.md
            let index_content = std::fs::read_to_string(ws.join("index.md")).unwrap_or_default();
            let index_lower = index_content.to_lowercase();

            // ── Index consistency: page-in-index-multiple-times ──
            let mut index_paths_seen: HashMap<String, usize> = HashMap::new();
            for line in index_content.lines() {
                // Extract paths from markdown links: [text](path) or bare paths
                if let Some(start) = line.find("](") {
                    let path_start = start + 2;
                    if let Some(end) = line[path_start..].find(')') {
                        let path = &line[path_start..path_start + end];
                        if path.starts_with("wiki/") {
                            *index_paths_seen.entry(path.to_string()).or_insert(0) += 1;
                        }
                    }
                }
            }
            for (path, count) in &index_paths_seen {
                if *count > 1 {
                    all_issues.push(LintIssue {
                        severity: "error".into(),
                        code: "page-in-index-multiple-times".into(),
                        path: "index.md".into(),
                        message: format!("path `{}` appears {} times in index.md", path, count),
                    });
                }
            }

            // Lint each wiki page
            for page_path in &all_pages {
                let content = std::fs::read_to_string(ws.join(page_path))?;
                let line_count = content.lines().count();

                // Frontmatter checks (required fields, type, tags, sources)
                all_issues.extend(check_frontmatter(page_path, &content));

                // Wikilink + orphan + page-size checks
                let parsed = parse_frontmatter(&content)?;
                let inbound_count = inbound.get(page_path).copied().unwrap_or(0);
                all_issues.extend(check_wikilinks(page_path, &parsed.body, &all_pages, inbound_count, line_count));

                // ── Wikilink additions: link-to-self, case-mismatch ──
                let links = extract_wikilinks(&parsed.body);
                let page_slug = Path::new(page_path).file_stem().and_then(|s| s.to_str()).unwrap_or("");
                for link in &links {
                    let link_slug = link.split('/').last().unwrap_or(link);
                    if link_slug.eq_ignore_ascii_case(page_slug) {
                        all_issues.push(LintIssue {
                            severity: "warn".into(),
                            code: "link-to-self".into(),
                            path: page_path.into(),
                            message: format!("page links to itself via `[[{}]]`", link),
                        });
                    } else if !all_pages.iter().any(|p| {
                        Path::new(p).file_stem().and_then(|s| s.to_str()).unwrap_or("") == link_slug
                    }) {
                        // Already caught as dangling-link by check_wikilinks
                    } else {
                        // Check case-mismatch: link exists but only with different case
                        let link_lower = link_slug.to_lowercase();
                        let exact_match = all_pages.iter().any(|p| {
                            Path::new(p).file_stem().and_then(|s| s.to_str()).unwrap_or("") == link_slug
                        });
                        if !exact_match {
                            all_issues.push(LintIssue {
                                severity: "warn".into(),
                                code: "case-mismatch".into(),
                                path: page_path.into(),
                                message: format!("`[[{}]]` matches only via case-insensitive lookup", link),
                            });
                        }
                    }
                }

                // ── Date checks: invalid-date, updated-before-created ──
                if let Some(fm) = parsed.frontmatter.as_mapping() {
                    let created = fm.get("created").and_then(|v| v.as_str());
                    let updated = fm.get("updated").and_then(|v| v.as_str());
                    let today = chrono::Utc::now().date_naive();

                    if let Some(c) = created {
                        if chrono::NaiveDate::parse_from_str(c, "%Y-%m-%d").is_err() {
                            all_issues.push(LintIssue {
                                severity: "warn".into(),
                                code: "invalid-date".into(),
                                path: page_path.into(),
                                message: format!("`created` value `{}` is not YYYY-MM-DD", c),
                            });
                        }
                    }
                    if let Some(u) = updated {
                        if chrono::NaiveDate::parse_from_str(u, "%Y-%m-%d").is_err() {
                            all_issues.push(LintIssue {
                                severity: "warn".into(),
                                code: "invalid-date".into(),
                                path: page_path.into(),
                                message: format!("`updated` value `{}` is not YYYY-MM-DD", u),
                            });
                        }
                    }
                    if let (Some(c), Some(u)) = (created, updated) {
                        if let (Ok(c_date), Ok(u_date)) = (
                            chrono::NaiveDate::parse_from_str(c, "%Y-%m-%d"),
                            chrono::NaiveDate::parse_from_str(u, "%Y-%m-%d"),
                        ) {
                            if u_date < c_date {
                                all_issues.push(LintIssue {
                                    severity: "warn".into(),
                                    code: "updated-before-created".into(),
                                    path: page_path.into(),
                                    message: format!("`updated` ({}) is before `created` ({})", u, c),
                                });
                            }
                        }
                    }

                    // Old page: updated > 6 months ago
                    if let Some(u) = updated {
                        if let Ok(u_date) = chrono::NaiveDate::parse_from_str(u, "%Y-%m-%d") {
                            if today - u_date > chrono::Duration::days(180) {
                                all_issues.push(LintIssue {
                                    severity: "warn".into(),
                                    code: "old-page".into(),
                                    path: page_path.into(),
                                    message: format!("page last updated {} ago (> 6 months)", today - u_date),
                                });
                            }
                        }
                    }

                    // Review_by overdue
                    if let Some(review_by) = fm.get("review_by").and_then(|v| v.as_str()) {
                        if let Ok(date) = chrono::NaiveDate::parse_from_str(review_by, "%Y-%m-%d") {
                            if date < today {
                                all_issues.push(LintIssue {
                                    severity: "warn".into(),
                                    code: "review-overdue".into(),
                                    path: page_path.into(),
                                    message: format!("review_by `{}` has passed", review_by),
                                });
                            }
                        }
                    }

                    // Low confidence flag
                    if let Some(conf) = fm.get("confidence").and_then(|v| v.as_str()) {
                        if conf == "low" {
                            all_issues.push(LintIssue {
                                severity: "warn".into(),
                                code: "low-confidence".into(),
                                path: page_path.into(),
                                message: "page has `confidence: low` — review before trusting".into(),
                            });
                        }
                    }

                    // Contested flag
                    if let Some(contested) = fm.get("contested") {
                        if contested.as_bool() == Some(true) {
                            all_issues.push(LintIssue {
                                severity: "warn".into(),
                                code: "contested".into(),
                                path: page_path.into(),
                                message: "page has `contested: true` — unresolved contradictions".into(),
                            });
                        }
                    }
                }

                // ── Footnote & citation checks ──
                let body = &parsed.body;
                // Collect footnote references: [^N] (not definitions)
                let mut refs_seen: Vec<(usize, String)> = vec![];
                // Collect footnote definitions: [^N]:
                let mut defs_seen: Vec<(usize, String)> = vec![];
                // Track if definitions are grouped at end
                let mut last_def_line = 0usize;
                let mut first_def_line = usize::MAX;
                let mut last_content_line = 0usize;
                let mut in_defs = false;

                for (i, line) in body.lines().enumerate() {
                    let trimmed = line.trim();
                    // Definition: [^N]: ...
                    if let Some(rest) = trimmed.strip_prefix("[^") {
                        if let Some(end) = rest.find("]:") {
                            let label = &rest[..end];
                            defs_seen.push((i + 1, label.to_string()));
                            last_def_line = i + 1;
                            if first_def_line > i + 1 { first_def_line = i + 1; }
                            in_defs = true;
                            continue;
                        }
                    }
                    if !trimmed.is_empty() {
                        last_content_line = i + 1;
                        if !trimmed.starts_with("[^") {
                            in_defs = false;
                        }
                    }
                    // Reference: [^N] (not followed by :)
                    if let Some(rest) = trimmed.find("[^") {
                        let after = &trimmed[rest + 2..];
                        if let Some(end) = after.find(']') {
                            let label = &after[..end];
                            // Ensure it's not a definition
                            if !trimmed.starts_with("[^") || !trimmed.contains("]:") {
                                refs_seen.push((i + 1, label.to_string()));
                            }
                        }
                    }
                }

                // footnote-undefined: reference used but no definition
                for (line_num, label) in &refs_seen {
                    if !defs_seen.iter().any(|(_, l)| l == label) {
                        all_issues.push(LintIssue {
                            severity: "error".into(),
                            code: "footnote-undefined".into(),
                            path: page_path.into(),
                            message: format!("line {}: `[^{}]` used but no definition found", line_num, label),
                        });
                    }
                }

                // footnote-unused: definition exists but never referenced
                for (line_num, label) in &defs_seen {
                    if !refs_seen.iter().any(|(_, l)| l == label) {
                        all_issues.push(LintIssue {
                            severity: "warn".into(),
                            code: "footnote-unused".into(),
                            path: page_path.into(),
                            message: format!("line {}: `[^{}]:` defined but never referenced", line_num, label),
                        });
                    }
                }

                // footnote-duplicate: same label defined twice
                let mut defs_labels: Vec<String> = defs_seen.iter().map(|(_, l)| l.clone()).collect();
                defs_labels.sort();
                defs_labels.dedup();
                if defs_labels.len() < defs_seen.len() {
                    all_issues.push(LintIssue {
                        severity: "error".into(),
                        code: "footnote-duplicate".into(),
                        path: page_path.into(),
                        message: "duplicate footnote labels found".into(),
                    });
                }

                // citation-unresolved: [^N]: points to file not in raw/
                for (_, label) in &defs_seen {
                    // Try to find the definition line content
                    if let Some(def_line) = body.lines().find(|l| l.trim().starts_with(&format!("[^{}]:", label))) {
                        let after_colon = def_line.trim().trim_start_matches(&format!("[^{}]:", label)).trim();
                        // If it looks like a file path (no spaces, has extension)
                        if after_colon.contains('/') && after_colon.contains('.') && !after_colon.contains(' ') {
                            let raw_target = ws.join(&after_colon);
                            if !raw_target.exists() {
                                all_issues.push(LintIssue {
                                    severity: "error".into(),
                                    code: "citation-unresolved".into(),
                                    path: page_path.into(),
                                    message: format!("`[^{}]:` points to `{}` which does not exist in raw/", label, after_colon),
                                });
                            }
                        }
                    }
                }

                // mid-document-footnotes: definitions not grouped at end
                if first_def_line < usize::MAX && last_def_line > 0 && last_content_line > last_def_line {
                    all_issues.push(LintIssue {
                        severity: "warn".into(),
                        code: "mid-document-footnotes".into(),
                        path: page_path.into(),
                        message: "footnote definitions not grouped at end of document".into(),
                    });
                }
            }

            // ── source-not-cited: raw files not referenced by any page ──
            let raw_dir = ws.join("raw");
            if raw_dir.exists() {
                for entry in walkdir::WalkDir::new(&raw_dir) {
                    if let Ok(e) = entry {
                        let ext = e.path().extension().and_then(|s| s.to_str()).unwrap_or("");
                        if ext != "md" { continue; }
                        let rel = e.path().strip_prefix(&ws).unwrap().to_string_lossy().replace('\\', "/");
                        if !referenced_raws.contains(&rel) {
                            all_issues.push(LintIssue {
                                severity: "warn".into(),
                                code: "source-not-cited".into(),
                                path: rel,
                                message: "raw file not cited by any wiki page".into(),
                            });
                        }
                    }
                }
            }
            for link_path in index_paths_seen.keys() {
                let target = ws.join(link_path);
                if !target.exists() {
                    all_issues.push(LintIssue {
                        severity: "error".into(),
                        code: "index-points-to-missing".into(),
                        path: "index.md".into(),
                        message: format!("index links to `{}` which does not exist", link_path),
                    });
                }
            }

            // ── Index consistency: index-no-section ──
            // Check that every page linked in index.md is under a ## Section
            let mut current_section: Option<String> = None;
            let mut page_has_section: HashSet<String> = HashSet::new();
            for line in index_content.lines() {
                if let Some(rest) = line.strip_prefix("## ") {
                    current_section = Some(rest.trim().to_string());
                } else if current_section.is_some() {
                    if let Some(start) = line.find("](") {
                        let path_start = start + 2;
                        if let Some(end) = line[path_start..].find(')') {
                            let path = &line[path_start..path_start + end];
                            if path.starts_with("wiki/") {
                                page_has_section.insert(path.to_string());
                            }
                        }
                    }
                }
            }
            for (path, _) in &index_paths_seen {
                if !page_has_section.contains(path) {
                    all_issues.push(LintIssue {
                        severity: "warn".into(),
                        code: "index-no-section".into(),
                        path: "index.md".into(),
                        message: format!("`{}` is not under any `## Section` in index.md", path),
                    });
                }
            }

            // ── Raw orphan detection: collect all sources referenced by wiki pages ──
            let mut referenced_raws: HashSet<String> = HashSet::new();
            for page_path in &all_pages {
                let content = std::fs::read_to_string(ws.join(page_path)).unwrap_or_default();
                if let Ok(parsed) = parse_frontmatter(&content) {
                    if let Some(fm) = parsed.frontmatter.as_mapping() {
                        if let Some(sources) = fm.get("sources").and_then(|v| v.as_sequence()) {
                            for src in sources {
                                if let Some(s) = src.as_str() {
                                    referenced_raws.insert(s.to_string());
                                }
                            }
                        }
                    }
                }
            }

            // ── Embedding checks ──
            let emb_path = ws.join("embeddings.jsonl");
            if emb_path.exists() {
                let emb_content = std::fs::read_to_string(&emb_path).unwrap_or_default();
                let emb_lines: Vec<&str> = emb_content.lines().filter(|l| !l.is_empty()).collect();

                if emb_lines.is_empty() && !all_pages.is_empty() {
                    all_issues.push(LintIssue {
                        severity: "warn".into(),
                        code: "embeddings-empty".into(),
                        path: "embeddings.jsonl".into(),
                        message: "wiki has pages but embeddings.jsonl is empty".into(),
                    });
                }

                let mut models_seen: HashSet<String> = HashSet::new();
                for line in &emb_lines {
                    if let Ok(val) = serde_json::from_str::<serde_json::Value>(line) {
                        if let Some(model) = val.get("model").and_then(|v| v.as_str()) {
                            models_seen.insert(model.to_string());
                        }
                        if let Some(path) = val.get("path").and_then(|v| v.as_str()) {
                            let target = ws.join(path);
                            if !target.exists() {
                                all_issues.push(LintIssue {
                                    severity: "warn".into(),
                                    code: "embeddings-orphan".into(),
                                    path: "embeddings.jsonl".into(),
                                    message: format!("embedding for `{}` — file no longer exists", path),
                                });
                            }
                        }
                    }
                }
                if models_seen.len() > 1 {
                    all_issues.push(LintIssue {
                        severity: "warn".into(),
                        code: "embeddings-model-mismatch".into(),
                        path: "embeddings.jsonl".into(),
                        message: format!("multiple embedding models: {}", models_seen.iter().collect::<Vec<_>>().join(", ")),
                    });
                }

                // embeddings-stale: compare page sha256 in frontmatter vs sha256 in embeddings.jsonl
                for line in &emb_lines {
                    if let Ok(val) = serde_json::from_str::<serde_json::Value>(line) {
                        if let (Some(path), Some(emb_sha)) = (
                            val.get("path").and_then(|v| v.as_str()),
                            val.get("sha256").and_then(|v| v.as_str()),
                        ) {
                            let page_file = ws.join(path);
                            if page_file.exists() {
                                if let Ok(content) = std::fs::read_to_string(&page_file) {
                                    if let Ok(parsed) = parse_frontmatter(&content) {
                                        if let Some(fm) = parsed.frontmatter.as_mapping() {
                                            if let Some(page_sha) = fm.get("sha256").and_then(|v| v.as_str()) {
                                                if page_sha != emb_sha {
                                                    all_issues.push(LintIssue {
                                                        severity: "warn".into(),
                                                        code: "embeddings-stale".into(),
                                                        path: "embeddings.jsonl".into(),
                                                        message: format!(
                                                            "`{}` page sha256 (`{}`) differs from embedding sha256 (`{}`) — re-embed",
                                                            path, &page_sha[..8.min(page_sha.len())], &emb_sha[..8.min(emb_sha.len())]
                                                        ),
                                                    });
                                                }
                                            }
                                        }
                                    }
                                }
                            }
                        }
                    }
                }
            }
        }
    }

    // ── Raw file lint ──
    if args.scope == "raw" || args.scope == "all" {
        let raw_dir = ws.join("raw");
        if raw_dir.exists() {
            for entry in walkdir::WalkDir::new(&raw_dir) {
                let e = entry?;
                let ext = e.path().extension().and_then(|s| s.to_str()).unwrap_or("");
                if ext != "md" { continue; }
                let rel = e.path().strip_prefix(&ws).unwrap().to_string_lossy().replace('\\', "/");
                let content = std::fs::read_to_string(e.path())?;
                let parsed = parse_frontmatter(&content)?;

                // raw-no-frontmatter: non-binary file lacks frontmatter
                if parsed.frontmatter.is_none() {
                    all_issues.push(LintIssue {
                        severity: "warn".into(),
                        code: "raw-no-frontmatter".into(),
                        path: rel.clone(),
                        message: "raw file lacks frontmatter block".into(),
                    });
                }

                if let Some(fm) = parsed.frontmatter.as_mapping() {
                    // Must have sha256 for drift detection
                    if let Some(declared_sha) = fm.get("sha256").and_then(|v| v.as_str()) {
                        // raw-drift: recompute sha256 of body and compare
                        let mut hasher = Sha256::new();
                        hasher.update(parsed.body.as_bytes());
                        let computed = format!("{:x}", hasher.finalize());
                        if computed != declared_sha {
                            all_issues.push(LintIssue {
                                severity: "error".into(),
                                code: "raw-drift".into(),
                                path: rel.clone(),
                                message: format!("sha256 drift: declared `{}` but body hashes to `{}`", &declared_sha[..16], &computed[..16]),
                            });
                        }
                    } else {
                        all_issues.push(LintIssue {
                            severity: "warn".into(),
                            code: "raw-no-sha256".into(),
                            path: rel.clone(),
                            message: "raw file missing `sha256` digest for drift detection".into(),
                        });
                    }

                    // Must have at least one source locator
                    let has_locator = fm.contains_key("source_url")
                        || fm.contains_key("session_url")
                        || fm.contains_key("source_path");
                    if !has_locator {
                        all_issues.push(LintIssue {
                            severity: "warn".into(),
                            code: "raw-missing-locator".into(),
                            path: rel.clone(),
                            message: "raw file missing source_url, session_url, or source_path".into(),
                        });
                    }
                }

                // raw-orphan: check if any wiki page references this raw file
                // (referenced_raws computed earlier in wiki lint block)
                // For standalone raw linting, we check after both blocks merge
            }
        }
    }

    // ── Log health ──
    let log_path = ws.join("log.md");
    if log_path.exists() {
        let log_content = std::fs::read_to_string(&log_path).unwrap_or_default();

        // log-bad-format: check entry format matches `## [YYYY-MM-DD] action | description`
        for (i, line) in log_content.lines().enumerate() {
            if line.starts_with("## ") {
                let entry = &line[3..];
                // Expected: [YYYY-MM-DD] action | description
                if !entry.starts_with('[') || !entry.contains("] ") {
                    all_issues.push(LintIssue {
                        severity: "warn".into(),
                        code: "log-bad-format".into(),
                        path: "log.md".into(),
                        message: format!("line {}: entry doesn't match `## [date] action | desc` format", i + 1),
                    });
                }
            }
        }

        // log-missing-entry: check if any wiki page modified in last 7 days has no log entry
        let today = chrono::Utc::now().date_naive();
        let log_text_lower = log_content.to_lowercase();
        if args.scope == "wiki" || args.scope == "all" {
            let wiki_dir = ws.join("wiki");
            if wiki_dir.exists() {
                for entry in walkdir::WalkDir::new(&wiki_dir) {
                    if let Ok(e) = entry {
                        if e.path().extension().and_then(|s| s.to_str()) == Some("md") {
                            let slug = e.path().file_stem().and_then(|s| s.to_str()).unwrap_or("");
                            if let Ok(meta) = e.metadata() {
                                if let Ok(modified) = meta.modified() {
                                    if let Ok(duration) = modified.duration_since(std::time::UNIX_EPOCH) {
                                        let days = duration.as_secs() / 86400;
                                        let file_date = chrono::NaiveDate::from_num_days_from_ce(days as i32 + 719163);
                                        if today - file_date < chrono::Duration::days(7) {
                                            if !log_text_lower.contains(&slug.to_lowercase()) {
                                                all_issues.push(LintIssue {
                                                    severity: "warn".into(),
                                                    code: "log-missing-entry".into(),
                                                    path: "log.md".into(),
                                                    message: format!("page `{}` modified recently but no log entry found", slug),
                                                });
                                            }
                                        }
                                    }
                                }
                            }
                        }
                    }
                }
            }
        }
    }

    // ── TODO lifecycle ──
    let todo_dir = ws.join("TODO");
    if todo_dir.exists() {
        let count = walkdir::WalkDir::new(&todo_dir)
            .into_iter()
            .filter_map(|e| e.ok())
            .filter(|e| e.path().extension().and_then(|s| s.to_str()) == Some("md"))
            .count();
        if count > 0 {
            all_issues.push(LintIssue {
                severity: "warn".into(),
                code: "todo-remaining".into(),
                path: "TODO/".into(),
                message: format!("{} items in TODO/ — clear after ingestion or log them", count),
            });
        }
    }

    // ── Query compounding ──
    if args.scope == "wiki" || args.scope == "all" {
        let index_content = std::fs::read_to_string(ws.join("index.md")).unwrap_or_default();
        if index_content.to_lowercase().contains("## queries") {
            let queries_section = index_content.split("## queries").nth(1).unwrap_or("");
            if queries_section.lines().filter(|l| l.trim().starts_with('-')).count() == 0 {
                all_issues.push(LintIssue {
                    severity: "warn".into(),
                    code: "queries-empty".into(),
                    path: "index.md".into(),
                    message: "queries section in index.md is empty — compound valuable answers into pages".into(),
                });
            }
        }
    }

    // ── Report ──
    let errors = all_issues.iter().filter(|i| i.severity == "error").count();
    let warnings = all_issues.iter().filter(|i| i.severity == "warn").count();

    if args.json {
        println!("{}", serde_json::to_string_pretty(&serde_json::json!({
            "errors": errors,
            "warnings": warnings,
            "issues": all_issues,
        })).unwrap());
    } else {
        println!("\n{} error(s), {} warning(s)\n", errors, warnings);
        for issue in &all_issues {
            println!("  [{}] [{}] {} — {}", issue.severity, issue.code, issue.path, issue.message);
        }
    }

    if errors > 0 { std::process::exit(2); }
    if args.strict && warnings > 0 { std::process::exit(2); }
    Ok(())
}

/// Build a map: page_path → number of inbound [[wikilinks]] from other pages.
fn compute_inbound_links(ws: &Path, all_pages: &[String]) -> HashMap<String, usize> {
    let mut inbound: HashMap<String, usize> = HashMap::new();
    for page_path in all_pages {
        let content = std::fs::read_to_string(ws.join(page_path)).unwrap_or_default();
        if let Ok(parsed) = parse_frontmatter(&content) {
            let links = extract_wikilinks(&parsed.body);
            for link in &links {
                let target_slug = link.split('/').last().unwrap_or(link).to_lowercase();
                for candidate in all_pages {
                    let candidate_slug = Path::new(candidate)
                        .file_stem()
                        .and_then(|s| s.to_str())
                        .unwrap_or("")
                        .to_lowercase();
                    if candidate_slug == target_slug && candidate != page_path {
                        *inbound.entry(candidate.clone()).or_insert(0) += 1;
                    }
                }
            }
        }
    }
    inbound
}
```

```rust
use std::path::PathBuf;
use crate::core::embeddings::EmbeddingsFile;
use crate::core::workspace::discover_workspace;
use crate::error::WikiError;

pub struct StatusArgs {
    pub workspace: Option<PathBuf>,
    pub json: bool,
}

pub fn run(args: StatusArgs) -> Result<(), WikiError> {
    let ws = discover_workspace(
        args.workspace.clone(),
        std::env::var("WIKI_WORKSPACE").ok().map(PathBuf::from),
        std::env::current_dir()?,
    )?;

    let mut page_count = 0;
    let wiki_dir = ws.join("wiki");
    if wiki_dir.exists() {
        for entry in walkdir::WalkDir::new(&wiki_dir) {
            let e = entry?;
            if e.path().extension().and_then(|s| s.to_str()) == Some("md") {
                page_count += 1;
            }
        }
    }

    let emb = EmbeddingsFile::read_from(&ws.join("embeddings.jsonl"))?;
    let embedded_pages: std::collections::HashSet<String> = emb.pages.iter().map(|p| p.path.clone()).collect();
    let mut total_chunks = 0;
    for p in &emb.pages { total_chunks += p.chunks.len(); }

    let raw_count = if ws.join("raw").exists() {
        walkdir::WalkDir::new(ws.join("raw")).into_iter().filter_map(|e| e.ok()).filter(|e| e.path().is_file()).count()
    } else { 0 };

    if args.json {
        println!("{}", serde_json::to_string_pretty(&serde_json::json!({
            "workspace": ws.display().to_string(),
            "pages": page_count,
            "embedded_pages": embedded_pages.len(),
            "embedding_chunks": total_chunks,
            "raw_sources": raw_count,
        })).unwrap());
    } else {
        println!("\nWiki: {}", ws.display());
        println!("  Pages: {}", page_count);
        println!("  Embedded: {} ({} chunks)", embedded_pages.len(), total_chunks);
        println!("  Raw sources: {}", raw_count);
    }
    Ok(())
}
```

Drop a source file into `raw/articles/` (or another `raw/` subdir), then:

wiki ingest raw/articles/filename.md

```markdown
# Troubleshooting

**`workspace not found`** — You're not in a wiki folder. Run `wiki --workspace /path/to/wiki <cmd>` or `cd ~/my-wiki` first.

**`NIM API key not set`** — `export NVIDIA_NIM_API_KEY="nvapi-..."` in your shell rc, or pass it inline.

**`NIM unreachable`** — Check `wiki doctor`. Common causes: bad API key (401), rate limit (429), network issue. The CLI retries with backoff on 5xx.

**`no embeddings yet`** — Run `wiki embed` first.

**`No matches for query`** — Try lowering `--threshold` (default 0.3), broadening the query, or running `wiki embed --skip-existing` to refresh stale vectors.

**`lint: missing-frontmatter`** — Add YAML frontmatter. The CLI can scaffold one with `wiki init` on a fresh page, or run `wiki lint --fix`.

**`file changed during embed`** — Re-run `wiki embed`. The JSONL is atomic on write, so partial state is impossible.

**`git push rejected`** — Likely a non-fast-forward. Run `git pull --rebase` then push.

**Embeddings slow on first run** — First embed re-computes everything from scratch. Subsequent runs use `--skip-existing` and only embed changed files (by sha256).

**Out of disk** — embeddings.jsonl is gitignored; clean it up with `rm embeddings.jsonl && wiki embed`. The wiki content is unaffected.

**Wrong model in embeddings** — Edit `nim.embed_model` in `.wiki/config.yaml` and re-run `wiki embed`. Old entries with previous models are kept (different model entries coexist in the JSONL).
```

```rust
use std::path::PathBuf;
use crate::error::WikiError;

pub struct InstallSkillArgs {
    pub global: bool,
    pub workspace: Option<PathBuf>,
    pub target: Option<PathBuf>,
}

pub fn run(args: InstallSkillArgs) -> Result<(), WikiError> {
    let source = args.target.unwrap_or_else(|| {
        // Default: relative to the binary's repo (find Cargo.toml)
        std::env::current_exe()
            .ok()
            .and_then(|p| p.parent().map(|p| p.to_path_buf()))
            .map(|p| p.join("agents/skills/wiki"))
            .unwrap_or_else(|| PathBuf::from("agents/skills/wiki"))
    });

    if !source.exists() {
        return Err(WikiError::Other(anyhow::anyhow!(
            "skill source not found: {}", source.display()
        )));
    }

    let target = if args.global {
        let home = std::env::var("HOME").map_err(|_| WikiError::Other(anyhow::anyhow!("HOME not set")))?;
        PathBuf::from(home).join(".agents/skills/wiki")
    } else {
        let ws = args.workspace.unwrap_or_else(|| std::env::current_dir().unwrap());
        ws.join(".agents/skills/wiki")
    };

    if let Some(parent) = target.parent() {
        std::fs::create_dir_all(parent)?;
    }
    if target.exists() || target.symlink_metadata().is_ok() {
        std::fs::remove_dir_all(&target).ok();
    }

    std::os::unix::fs::symlink(&source, &target)?;
    println!("✓ Installed skill: {} -> {}", target.display(), source.display());
    Ok(())
}
```

```rust
use std::path::{Path, PathBuf};
use sha2::{Sha256, Digest};
use chrono::Utc;
use crate::core::workspace::discover_workspace;
use crate::error::WikiError;

pub struct IngestArgs {
    pub workspace: Option<PathBuf>,
    pub source: PathBuf,
    pub no_compile: bool,
    pub source_type: Option<String>,
}

pub fn run(args: IngestArgs) -> Result<(), WikiError> {
    let ws = discover_workspace(
        args.workspace.clone(),
        std::env::var("WIKI_WORKSPACE").ok().map(PathBuf::from),
        std::env::current_dir()?,
    )?;

    if !args.source.exists() {
        return Err(WikiError::Other(anyhow::anyhow!("source not found: {}", args.source.display())));
    }

    let filename = args.source.file_name().unwrap().to_string_lossy().to_string();
    let dest = ws.join("raw/articles").join(&filename);

    let body = std::fs::read_to_string(&args.source)?;
    let sha = format!("{:x}", Sha256::digest(body.as_bytes()));
    let today = Utc::now().format("%Y-%m-%d").to_string();

    let frontmatter = format!(
        "---\nsource_type: {}\ningested: {}\nsha256: {}\n---\n\n",
        args.source_type.clone().unwrap_or_else(|| "article".into()),
        today, sha
    );

    std::fs::write(&dest, format!("{}{}", frontmatter, body))?;
    println!("✓ Added source to {}", dest.display());

    // Append to log
    let log_path = ws.join("log.md");
    let log_entry = format!("\n## [{}] ingest | {}\n- Added source: {}\n", today, filename, dest.strip_prefix(&ws).unwrap().display());
    let mut log = std::fs::read_to_string(&log_path).unwrap_or_else(|_| "# Log\n".into());
    log.push_str(&log_entry);
    std::fs::write(&log_path, log)?;

    if !args.no_compile {
        println!("Note: LLM-driven compile pass is not yet implemented in v1. Run `wiki build` to trigger it.");
    }

    Ok(())
}
```

```typescript
export async function loadPages(): Promise<PageMeta[]> {

  if (!resp.ok) throw new Error(`Failed to load pages: ${resp.status}`);
  return resp.json();
}

  if (!resp.ok) throw new Error(`Failed to load embeddings: ${resp.status}`);
  return resp.json();

// ── Offline brute-force cosine search ──
// The query embedding is computed client-side via a tiny inline model,
// OR the caller passes a pre-computed embedding vector.
// For MVP: rank by BM25-style keyword overlap (no embedding needed at query time).
// This avoids requiring a browser-side ONNX runtime.

export function searchPages(query: string, pages: PageMeta[]): SearchResult[] {
  const tokens = tokenize(query);
  if (tokens.length === 0) return [];

  const scores: SearchResult[] = pages.map((p) => {
    const titleTokens = tokenize(p.title);
    const contentTokens = tokenize(p.content_html);
    const tagTokens = p.tags.map(t => t.toLowerCase());

    let score = 0;
    for (const t of tokens) {
      // Title match: high weight
      if (titleTokens.some(tt => tt.includes(t) || t.includes(tt))) score += 3;
      // Tag match: medium weight
      if (tagTokens.some(tt => tt.includes(t) || t.includes(tt))) score += 2;
      // Content match: low weight
      if (contentTokens.some(ct => ct.includes(t))) score += 1;
    }

    return { slug: p.slug, score, title: p.title };
  });

  return scores
    .filter(r => r.score > 0)
    .sort((a, b) => b.score - a.score)
    .slice(0, 20);
}

export function cosineSimilarity(a: number[], b: number[]): number {
  if (a.length !== b.length) return 0;
  let dot = 0, na = 0, nb = 0;
  for (let i = 0; i < a.length; i++) {
    dot += a[i] * b[i];
    na += a[i] * a[i];
    nb += b[i] * b[i];
  }
  return dot / (Math.sqrt(na) * Math.sqrt(nb));
}

function tokenize(text: string): string[] {
  return text
    .toLowerCase()
    .replace(/[^a-z0-9\s-]/g, ' ')
    .split(/\s+/)
    .filter(t => t.length > 2);
}
```

```rust
use std::collections::HashMap;
use std::path::{Path, PathBuf};
use crate::core::markdown::{parse_frontmatter, extract_wikilinks};
use crate::error::WikiError;
use super::{PageMeta, EmbeddingEntry, GraphData, GraphNode, GraphEdge};

/// Generate _pages.json, _embeddings.json, _graph.json into `dest` directory.
pub fn build_viewer_json(ws: &Path, dest: &Path) -> Result<(), WikiError> {
    let wiki_dir = ws.join("wiki");
    if !wiki_dir.exists() {
        return Err(WikiError::Other(anyhow::anyhow!("wiki/ directory not found")));
    }

    // 1. Collect all wiki pages
    let mut page_paths: Vec<PathBuf> = vec![];
    for entry in walkdir::WalkDir::new(&wiki_dir) {
        let e = entry?;
        if e.path().extension().and_then(|s| s.to_str()) == Some("md") {
            page_paths.push(e.path().to_path_buf());
        }
    }

    // 2. Build slug → backlinks map (inbound links from other pages)
    let mut backlinks: HashMap<String, Vec<String>> = HashMap::new();
    let mut outbound_map: HashMap<String, Vec<String>> = HashMap::new();

    for page_path in &page_paths {
        let content = std::fs::read_to_string(page_path)?;
        if let Ok(parsed) = parse_frontmatter(&content) {
            let slug = page_path.file_stem().and_then(|s| s.to_str()).unwrap_or("").to_string();
            let links = extract_wikilinks(&parsed.body);
            outbound_map.insert(slug.clone(), links.iter().map(|l| {
                l.split('/').last().unwrap_or(l).to_string()
            }).collect());
            for link in &links {
                let target_slug = link.split('/').last().unwrap_or(link).to_string();
                backlinks.entry(target_slug).or_default().push(slug.clone());
            }
        }
    }

    // 3. Build PageMeta array
    let mut pages: Vec<PageMeta> = vec![];
    for page_path in &page_paths {
        let content = std::fs::read_to_string(page_path)?;
        let line_count = content.lines().count();
        let parsed = parse_frontmatter(&content)?;

        let slug = page_path.file_stem().and_then(|s| s.to_str()).unwrap_or("").to_string();
        let title = parsed.frontmatter.as_ref().and_then(|fm| {
            fm.as_mapping()?.get("title")?.as_str().map(String::from)
        }).unwrap_or_else(|| slug.clone());

        let r#type = parsed.frontmatter.as_ref().and_then(|fm| {
            fm.as_mapping()?.get("type")?.as_str().map(String::from)
        }).unwrap_or_else(|| "concept".into());

        let tags = parsed.frontmatter.as_ref().and_then(|fm| {
            fm.as_mapping()?.get("tags")?.as_sequence()?.iter()
                .filter_map(|v| v.as_str().map(String::from))
                .collect::<Vec<_>>().into()
        }).unwrap_or_default();

        let sources = parsed.frontmatter.as_ref().and_then(|fm| {
            fm.as_mapping()?.get("sources")?.as_sequence()?.iter()
                .filter_map(|v| v.as_str().map(String::from))
                .collect::<Vec<_>>().into()
        }).unwrap_or_default();

        let updated = parsed.frontmatter.as_ref().and_then(|fm| {
            fm.as_mapping()?.get("updated")?.as_str().map(String::from)
        }).unwrap_or_default();

        let confidence = parsed.frontmatter.as_ref().and_then(|fm| {
            fm.as_mapping()?.get("confidence")?.as_str().map(String::from)
        });

        let contested = parsed.frontmatter.as_ref().and_then(|fm| {
            fm.as_mapping()?.get("contested")?.as_bool()
        });

        let slug_lower = slug.to_lowercase();
        let page_backlinks = backlinks.get(&slug_lower).cloned().unwrap_or_default();
        let page_outbound = outbound_map.get(&slug_lower).cloned().unwrap_or_default();

        pages.push(PageMeta {
            slug,
            title,
            r#type,
            tags,
            sources,
            updated,
            content_html: parsed.body.clone(),
            backlinks: page_backlinks,
            outbound: page_outbound,
            line_count,
            confidence,
            contested,
        });
    }

    // 4. Build graph
    let mut graph_nodes: Vec<GraphNode> = vec![];
    let mut graph_edges: Vec<GraphEdge> = vec![];

    for page in &pages {
        graph_nodes.push(GraphNode {
            id: page.slug.clone(),
            title: page.title.clone(),
            r#type: page.r#type.clone(),
            inbound: page.backlinks.len(),
            outbound: page.outbound.len(),
        });
        for target in &page.outbound {
            graph_edges.push(GraphEdge {
                source: page.slug.clone(),
                target: target.clone(),
            });
        }
    }

    // 5. Load embeddings if embeddings.jsonl exists
    let embeddings: Vec<EmbeddingEntry> = {
        let emb_path = ws.join("embeddings.jsonl");
        if emb_path.exists() {
            let content = std::fs::read_to_string(&emb_path)?;
            content.lines()
                .filter_map(|line| serde_json::from_str(line).ok())
                .collect()
        } else {
            vec![]
        }
    };

    // 6. Write JSON files
    std::fs::create_dir_all(dest)?;

    let pages_json = serde_json::to_string_pretty(&pages)?;
    std::fs::write(dest.join("_pages.json"), &pages_json)?;

    let embeddings_json = serde_json::to_string_pretty(&embeddings)?;
    std::fs::write(dest.join("_embeddings.json"), &embeddings_json)?;

    let graph = GraphData { nodes: graph_nodes, edges: graph_edges };
    let graph_json = serde_json::to_string_pretty(&graph)?;
    std::fs::write(dest.join("_graph.json"), &graph_json)?;

    println!(
        "Viewer JSON built: {} pages, {} embeddings, {} graph nodes",
        pages.len(), embeddings.len(), graph.nodes.len()
    );

    Ok(())
}
```

### Naming Clarifications

- `src/viewer/build.rs` is a **Rust module file** (`mod build`), NOT the Cargo build script (`build.rs` at crate root). There is no Cargo build script in this project.
- The `build_viewer_json` function lives in `src/viewer/build.rs` and produces `_pages.json`, `_embeddings.json`, `_graph.json`.
- The `$lib/data/api` path in Task 30 refers to `src/lib/data/api.ts` — a TypeScript module that fetches `/_pages.json` via HTTP, NOT a direct data import.

### Edge Cases to Handle During Implementation

1. **Empty wiki search**: `wiki search` on a workspace with no embeddings.jsonl should error with: "No embeddings found. Run `wiki embed` first."
2. **Missing wiki/ directory**: Commands that read wiki/ (search, query, lint, doctor, status) should detect and report a clear error if wiki/ doesn't exist.
3. **Corrupted embeddings.jsonl**: A malformed line in embeddings.jsonl should be skipped with a warning, not crash the entire operation.
4. **Largest embeddings file**: `_embeddings.json` loaded in browser — 1000 pages × 2048-dim float32 = ~8MB. Acceptable for single-developer tool.
5. **Concurrent wiki embed**: No locking. Two simultaneous embed processes could corrupt embeddings.jsonl. Document as known limitation; atomic rename mitigates but does not fully prevent.

### Lint Implementation Notes

- Footnote checks (undefined/unused/duplicate) depend on `extract_footnotes` and `extract_footnote_refs` from Task 9. Import via `crate::core::markdown`.
- Citation checks use regex for source extraction (`\[([A-Za-z][^\]]*)\]`) — acceptable for v1, not a full citation parser.
- `--format json` and `--fix` are NOT implemented in Task 21. They're future enhancements. The plan covers only `--strict` and `--verbose` flags.
- The `embeddings-model-mismatch` check reads the config file and compares the embedding model against the first entry in embeddings.jsonl. This is a sample check, not a full file scan.
- Log checks parse `raw/ingestion.log` line-by-line. Each line is `ISO8601|slug|status|sha256`. A bad-format line should be reported as a warning, not crash the linter.
